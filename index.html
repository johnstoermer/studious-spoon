Neon Stick Shooter

Geometry Dash–style chaos for 2–4 players

Controls

  • Move: WASD (P1), Arrow keys (P2), IJKL (P3), Numpad (P4)
  • Jump: W / Up / I / Numpad8 (double-jump enabled)
  • Crouch/Slide: S / Down / K / Numpad5
  • Throw weapon: Q / Right Ctrl / U / Numpad0
  • Mouse: Aim & shoot (primary fire)
